# Student Registration Form

A simple **Student Registration Form** developed using **HTML** as part of the **Git and GitHub Portfolio Setup Assignment**.

## Project Description

This project is a basic web form that allows students to enter their registration details. The purpose of this project is to demonstrate the use of Git and GitHub for version control and project management.

## Features

* Student Name input field
* Email input field
* Phone Number input field
* Gender selection using radio buttons
* Course selection using a dropdown menu
* Submit button

## Technologies Used

* HTML5
* Git
* GitHub
